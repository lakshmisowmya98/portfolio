Retail Sales Analytics Using SQL

Advanced SQL project using CTEs, Window Functions, Ranking Functions, Customer Segmentation and Pareto Analysis.

Project Overview

This project analyzes Superstore sales data using advanced SQL techniques. The objective was to identify top-performing customers, profitable products, regional sales trends and revenue-driving products through business-focused SQL analysis.

Tools & Skills Used

SQL SQLite CTE Window Functions RANK() Business Analytics Customer Segmentation Pareto Analysis

Top Customers by Region

Used Common Table Expressions (CTEs) and Window Functions to rank customers based on revenue contribution within each region. This helps identify the highest-value customers for targeted business strategies.

Customer Segmentation Analysis

Categorized customers into VIP, Premium and Regular segments based on total sales contribution. This type of segmentation helps businesses prioritize retention and loyalty programs.

Product Profitability Analysis

Evaluated revenue, profit and profit margin percentage for products to identify highly profitable items and support pricing and inventory decisions.

Regional Performance Analysis

Compared sales revenue, profitability and customer counts across different regions to determine the strongest and weakest performing markets.

Pareto (80/20) Analysis

Applied cumulative revenue analysis using Window Functions to identify products contributing the majority of total sales. This supports portfolio optimization and revenue planning.
